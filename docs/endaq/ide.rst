#############
``endaq.ide``
#############

The submodule ``endaq.ide`` contains utility functions for accessing data in *Mide Instrumentation Data Exchange*
(``.IDE``) files, the native format generated by enDAQ data recorders.

A collection of :doc:`usage examples <ide>` is available.
